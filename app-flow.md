# App Flow

## Watch

### Idle
- Shows last job (optional)
- Start button available

### Active Shift
- Timer running
- Job displayed
- Voice note button available anytime

### Voice Note
- Tap microphone
- Record audio
- Auto-attached to current shift

### Stop Shift
- Confirm stop
- Shift marked complete
- Await sync

---

## Phone

- Receives shift + audio metadata
- Stores locally
- Optional transcription
- Export/share
